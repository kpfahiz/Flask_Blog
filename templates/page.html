NETWORK HACKING - POST CONNECTION ATTACK - INFORMATION GATHERING

i)Discovering Device Connected to the same Network

                Discover all device on their on network.
                Display their:
            
                        - IP Address
                        - MAC Address
                        - OS
                        - Open Ports
                        - Running Service
                        - etc......
            
                There are different tools used for discovering information about 
                target  devices such as NetDiscover & Nmap .
            
            Example with NetDiscover:
            
                Here used to details of wired connections:
            
                    netdiscover -r 10.0.2.1/24
                      
                       r                     - range of the IP Address.
                       10.0.2.1/24    - It tells range 10.0.2.1 to 10.0.2.254
            
                Discover details of wireless connections :
            
                     netdiscover -r 192.168.1.1/24
            NOTE :
                        To work this command on wireless network you  should disable
                        NAT network.
            
            ii)Gathering Sensitive Information About Connected Devices
                Here we are using Nmap/Zmap to gathering sensitive information about
                 connected devices such as MAC address , open port, running services,OS
                 ,connected clients ,etc...
